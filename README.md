# mftail

[mftail](https://github.com/deadc0de6/mftail) is multiple files follower (similar to `tail -f`) with color.

# Usage

usage
```bash
mftail v0.1

Usage: mftail <path>...
  -v	Print version.
  -version
    	Print version.
```

# Compile from source

compile
```bash
go build
mftail -help
```
